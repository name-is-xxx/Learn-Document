# 各类项目创建方式

### vue2/3 (使用 vue-cli)

1. 安装 vue cli
   1. 使用 npm
   ```
   npm install -g @vue/cli
   ```
   2. 使用 yarn
   ```
   yarn global add @vue/cli
   ```
2. 创建项目

```
vue create project-name
```

### React

1. 免安装下载

```
npx create-react-app project-name
```

2. 安装下载
   1. npm 下载
   ```
   npm i -g create-react-app
   create-react-app your-app
   ```
